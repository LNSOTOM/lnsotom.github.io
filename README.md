# HySpec Arid Polar Ecosystems Project Page

This workspace contains a static research project page inspired by the interaction style of Nerfies-based project sites.

## Files

- `index.html`: main single-page site with three tabs.
- `static/css/index.css`: page styling.
- `static/js/index.js`: tab switching logic.

## PDF hookup

Place the PDF at `static/pdfs/hyspec_arid_polar_ecosystemsCNN.pdf`.

## Preview

You can serve the site locally with:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
